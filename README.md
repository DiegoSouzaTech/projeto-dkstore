# 🛍️ DKStore - Experiência Web Interativa

Uma vitrine digital moderna e imersiva, desenvolvida para oferecer uma experiência de compra fluida e visualmente atraente. Este projeto foca no equilíbrio entre estética refinada (UI) e usabilidade intuitiva (UX).

## Linguagens utilizadas

O projeto foi construído utilizando as tecnologias fundamentais da web moderna:

- **HTML5:** Estrutura semântica e acessível.
- **CSS3:** Estilização avançada, utilizando Flexbox/Grid e animações personalizadas.
- **JavaScript:** Lógica de interatividade, manipulação de DOM e efeitos dinâmicos.
